# GO-AdminPro
GO - 後台管理

## 權限關係總覽圖：
- 管理員可以自由添加或刪除對應角色<br />
- 角色可以自由添加或刪除對應權限<br />
- 可以單獨為某個管理員添加指定權限<br />
